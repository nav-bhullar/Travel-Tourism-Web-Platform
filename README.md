# Travel & Tourism Web Platform

A responsive travel website developed using HTML5, CSS3, and JavaScript. The platform allows users to explore travel destinations, view tourist attractions, register for access, and submit inquiries through a contact form.

## Features

- Responsive website design
- Home, About, Attractions, Contact, and Registration pages
- User registration using JavaScript
- Restricted access to travel attractions for unregistered users
- Form validation for registration and contact forms
- Travel package details and estimated costs
- Client-side authentication using Local Storage

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API

## How to Run

1. Download or clone the project.
2. Open `index.html` in a web browser.
3. Register a user account.
4. Access the Attractions page to view protected travel information.
